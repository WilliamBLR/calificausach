# Califica Profesores USACH — v10 (Vite + React + Tailwind)

## Requisitos
- Node.js 18+ y npm

## Ejecutar en modo desarrollo
```bash
npm install
npm run dev
```
Abre la URL que aparece (por ejemplo, `http://localhost:5173`).

## Producción (build)
```bash
npm run build
npm run preview
```
